Programming on Solana when you’ve just installed Linux and don’t know what an npm is…

Alright so to start off this tutorial is for anyone brand new to programming and just thinks Solana is cool and is like “Yes, now I want to learn programming”. Welcome.

Understanding how programs written in Rust on Solana work is really challenging, but you just wanna see what all the fuss is about. So you go to the tutorials and you’re like “wait a sec, I dunno how to use GitHub.”

This tutorial covers the stuff that comes before most of the programming tutorials out there, stuff that most developers find obvious, but takes hours of pain to figure out the first time.

This tutorial assumes a debian-based linux system (Ubuntu, etc.)

If you’re completely new, check out Pop_OS. It’s very user friendly and comes with a lot of useful stuff already installed.

Once you’ve got your OS installed and booted up, the first place to go is the terminal.

Check out this tutorial to understand the terminal better

The next thing we’ll need is an editor for our code. Visual Studio code is a good option and we’ll install that now. Later on I would recommend checking out Vim, which is a terminal-based editor. (typing “vim” in the terminal will give install options. After it installs, type “vimtutor” to learn more.)

To install VS Code, download the .deb file from their website.

  • Once downloaded, type “sudo apt install <path to your file>”
  • The command for mine which was in the Downloads folder, looked like “sudo apt install ./Downloads/code_1.61.2-1634656828_amd64.deb”
  • “./” represents the directory you are currently in.
  • If I “cd” into Downloads first my command would be “sudo apt install ./code_1.61.2-1634656828_amd64.deb”

Next is learning git and setting up a Github account.

  • Start here with this tutorial for an overview of git and Github.

Now that we’re setup on Github it is time to fire up some code!

Go to the solana-labs/example-helloworld repository

Fork and clone the repo. Follow the steps here

Now that we have a local copy, start looking at the README.md. The first step is installing dependencies, the software needed to run our code.

Installing node is first.

  • We will use “node version manager” or “nvm”.
  • Go here, scroll down and check out the readme.
  • Under “Install & Update” run one of the two scripts (try curl first).If it says “command not found” use the command it suggests to install curl.
  • Scroll down to “Verify Installation” and do that. If it outputs “nvm” you’re good. Otherwise try closing and reopening the terminal window like it says.
  • To install node type “nvm install 14”. This will install node 14.18.1, as well as “node package manager” or “npm”.

Check your versions with “node -v” and “npm -v” commands in the terminal.

Next is installing Rust.

This is pretty straightforward. Follow the excellent getting started page here.

Finally, install the solana command line tool suite.

Follow the instructions under the “MacOS and Linux” heading. Once typing “solana --version” is working you’re ready to go!

I'll leave it here for now. This is everything you need to get going on the first line of "solana-labs/example-helloworld" readme. Have fun!
